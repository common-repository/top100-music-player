== Description ==

A simple http://widget.top100.cn music player widget plugin for wordpress,you can add a flash music player to your wordpress site with a customized legal music list at http://widget.top100.cn .

It's only for the http://widget.top100.cn/, if you used it in wrong way, your computer will bang~ !

这是一个top100.cn网站音乐播放器插件，它可以在你的worpress上插入一个播放器，播放你在这个网站上订制的音乐播放列表. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `top100_music_player` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Turn to the Themes->Widgets Menu,drag the "Music" widget into the widget area you want placed.
4. Paste code generated in http://widget.top100.cn into widget setting form, and don't forget about "save", then refresh your wordpress, you will find it appeared.

e.g.

1. 上传top100_music_player目录到插件目录 `/wp-content/plugins/` 下；
2. 在后台管理页面，“插件“菜单下，启用插件；
3. 在”主题->小工具“菜单下，拖动”Music“widget到右边你想放置播放器插件的位置；
4. 在widget设置栏里，粘贴在widget.top100.cn上生成的代码，保存，刷新wordpress就会看到播放器出现；

== Frequently Asked Questions ==

= Why does it unworded ? =

仅仅是为播放 http://widget.top100.cn/ 上生成的播放列表而编写的，粘贴别的代码不一定好使.
